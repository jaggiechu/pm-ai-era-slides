# UCLA MBA Slides

Presentation built with the [frontend-slides](https://github.com/zarazhangrui/frontend-slides) skill.

## Files

- `presentation.html` — final slide deck (open in browser)
- `assets/` — images, logos, charts

## Navigation

- **Arrow keys** or **Space** — next/previous slide
- **Click nav dots** — jump to slide
- **Scroll / swipe** — on touch devices

## Customization

- Colors: edit `:root` CSS variables at the top of `presentation.html`
- Fonts: swap the Google Fonts / Fontshare `<link>` tag
- Animations: elements with `.reveal` class animate on entry

## Edit in Browser (if enabled)

- Hover top-left corner or press **E** to enter edit mode
- Click any text to edit inline
- **Ctrl+S** to save changes to localStorage
